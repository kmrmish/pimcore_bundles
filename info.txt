Commonly used commands : -----------------------------------------

CREATE DATABASE pimcore_bundles charset=utf8mb4;

SET GLOBAL innodb_file_format=Barracuda;
ALTER DATABASE pimcore_bundles CHARACTER SET = utf8mb4 COLLATE = utf8mb4_unicode_ci;
SET GLOBAL innodb_large_prefix = ON;

bin/console pimcore:deployment:classes-rebuild

php bin/console ecommerce:indexservice:bootstrap --create-or-update-index-structure
php bin/console ecommerce:indexservice:bootstrap --update-index

php bin/console assets:install public --symlink


grant all privileges on mydb.* to myuser@'%' identified by 'mypasswd';



Create a new bundle : -----------------------------------------
bin/console pimcore:generate:bundle
go to admin and extensions section and enable the bundle
remove cache and give chmod and chown permissions
go to src/ClassFields/ImageBundle/Resources/public/js/pimcore/startup.js
and uncomment the alert line. 
now reload the instance and check if alert comes
